# SEO Fix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Get english-coach.online indexed by Google and Yandex by pre-rendering static HTML for all 3 routes and adding the missing crawl infrastructure.

**Architecture:** Install `vite-ssg` to replace `ReactDOM.createRoot` with a `ViteSSG` export that pre-renders all routes to static HTML at build time. Cloudflare Pages serves these files directly — crawlers read full content without executing JS. React hydrates normally on the client.

**Tech Stack:** React 18, TypeScript, Vite 6, react-router-dom 6, react-helmet-async (already installed), vite-ssg (to install), Cloudflare Pages.

## Task 6: Update App.tsx to be the layout component

**Files:**
- Modify: `src/App.tsx`

- [ ] **Step 1: Replace Routes with Outlet and add providers**

Replace the entire contents of `src/App.tsx` with:
```tsx
import React from 'react'
import { Outlet } from 'react-router-dom'
import { HeroUIProvider, ToastProvider } from '@heroui/react'
import { HelmetProvider } from 'react-helmet-async'
import { useTranslation } from 'react-i18next'

const App: React.FC = () => {
  const { i18n } = useTranslation()

  React.useEffect(() => {
    document.documentElement.lang = i18n.language
  }, [i18n.language])

  const [renderKey, setRenderKey] = React.useState(0)

  React.useEffect(() => {
    const handleLanguageChange = () => setRenderKey(k => k + 1)
    window.addEventListener('languageChanged', handleLanguageChange)
    return () => window.removeEventListener('languageChanged', handleLanguageChange)
  }, [])

  return (
    <HelmetProvider>
      <HeroUIProvider>
        <ToastProvider />
        <React.Fragment key={renderKey}>
          <Outlet />
        </React.Fragment>
      </HeroUIProvider>
    </HelmetProvider>
  )
}

export default App
```

> **What changed:** `<Routes>/<Route>` removed — vite-ssg provides the router and renders the matched page via `<Outlet />`. Providers moved here from main.tsx. Unused `t` import removed. `renderKey` initialised to `0` instead of `Date.now()` (SSG-safe).
